Bouncing Window

Bouncing Window is a window that allows the user to throw a red square around their screen. Programmed in C++ using the wxWidgets library. The biggest challenge was getting the window to rotate. I solved this by creating bitmaps representing different angles, which could be used to change the window's shape, and swapped between them depending on the window's angle. In the future I want to clean up the code, and possibly remake the project entirely. I also want to improve the rotational physics.

FPS

FPS Game is my entry into 3D graphics. Programmed in C++ with the help of raylib. Biggest challenge was refreshing my linear algebra knowledge for working in 3D, especially with the camera. Want to add mode game-like features like upgrades and weapons if there is a lull in my schoolwork or over the winter. Also want to make the graphics look better, currently enrolled in computer graphics, so hopefully I learn helpful information for this project.

Wordle

Wordle Clone is a clone of the game Wordle created using JavaFx for a school project. Although the professor asked us to create it using SceneBuilder, I decided to program all of the GUI elements manually at the advice of this article by Dave Barrett. This was difficult, but it gave me a far greater understanding of how GUI elements are created and interact, and sparked my interest in creating more graphical applications.

Pong

Pong was my first introduction to OpenGL. I made this after learning OpenGl on learnopengl.com written by Joey de Vries. The biggest challenge was getting OpenGL, GLFW, and GLAD running and linked to my project. Ultimately, this project was more about getting more comfortable working with libraries in C++ than the actual game itself. I am still proud of my AI enemy paddle who predicts the movement of the ball and positions itself to create an extremely difficult return. I haven't been able to score even a single point against it.

Mouse Distance Tracker

Mouse Distance Tracker was a fun, quick project I created to track how far, in meters, my mouse moves on my screen. Created using Python and the tkinter, pyautogui and screeninfo packages.

Paint

Paint was a school project for computer graphics where we utilized OpenGL to create a drawing application. The point of the project was to gain an understanding of how the frame buffers of OpenGL work, and how user input is handled. Includes a way to change the color of the brush, background, as well as different brush shapes and types.
